# pdev-jay/plugins

pdev-jay의 Claude Code 플러그인 카탈로그(마켓플레이스). 플러그인 코드는 각자의 repo에 있고, 이 repo는 그 목록만 정의한다.

## 설치

```
/plugin marketplace add pdev-jay/plugins
/plugin install <plugin>@pdev-jay
```

## 수록 플러그인

| 플러그인 | repo | 설명 |
|---|---|---|
| `ratchet` | [pdev-jay/ratchet](https://github.com/pdev-jay/ratchet) | Floor-first agentic coding workflow — intent-before-code / verify-before-done 게이트. |
| `vault-driven-development` | [pdev-jay/vault-driven-development-plugin](https://github.com/pdev-jay/vault-driven-development-plugin) | per-project Source-of-Truth 패턴. |
| `flow-driven-development` | [pdev-jay/flow-driven-development](https://github.com/pdev-jay/flow-driven-development) | 바꾸려는 것의 FLOW를 따라 계획하는 workflow. plan → build → review → done. |
