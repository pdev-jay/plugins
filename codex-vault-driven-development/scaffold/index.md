## Quick reference

Deterministic tools run via the `vdd` dispatcher (on PATH from the plugin's `bin/` — no per-project copy).

| Command | Effect |
|---|---|
| `vdd lint` | Validate + auto rollup |
| `vdd map` | Render the broadcast/intent graph (Mermaid; `--orphans` = zero-reactor audit) |
| `vdd bootstrap` | Extract layers from codebase (LLM-assisted) |
| `vdd impact <file>` / `vdd blast <key>` | Reverse (code→pages) / forward (key→reactors) impact |
| `grep -rln "<layer>/_state-contract#<key>" docs/vault/` | Direct grep for reactors of a signal |
